# Landing — Thank You

Статична сторінка подяки після покупки (Meta Pixel вже вбудований).

## Структура

- `index.html` — головна сторінка (Vercel віддає її автоматично)
- `body-flow-hero.png` — зображення курсу
- `vercel.json` — конфігурація деплою

## Як задеплоїти

### 1. Завантажити на GitHub
1. Створи новий репозиторій на [github.com/new](https://github.com/new)
2. Завантаж усі файли з цієї папки (можна перетягнути через **Add file → Upload files**)
3. Зроби commit

### 2. Задеплоїти у Vercel
1. Зайди на [vercel.com/new](https://vercel.com/new)
2. Імпортуй свій GitHub-репозиторій
3. Framework Preset: **Other** (це звичайна статика, build не потрібен)
4. Натисни **Deploy**

Готово — сторінка буде доступна за адресою `your-project.vercel.app`.
